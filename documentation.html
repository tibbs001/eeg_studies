EEG-related Clinical Trials


The information provided on this page was obtained from ClinicalTrials.gov. The ClinicalTrials.gov advanced query feature was used to identify which studies the National Libary of Medicine (NLM) has identified as being linked to the keyword 'EEG'. On May 22, 2016, the NLM's advanced query tool returned 1,220 such studies. The NCT IDs that uniquely identify these studies were then used to retrieve information from a relational database of aggregated clinical trial data that has been developed and is supported by the Clinical Trials Transformation Initiative's (CTTI). This relational database is the Aggregate Analysis of ClincalTrials.gov (AACT) system.


Note: The AACT system is currently undergoing a significant upgrade. AACT 2.0 beta database is being hosted in the cloud and is available for review. If you are interested in providing suggestions or participating in the effort to make ClinicalTrials.gov data more accessible, please contact us at...... as a postgres database on Heroku and is available for use under the MIT software license. You can report bugs and discuss features on the GitHub issues page, or send tweets to CTTI-AACT2.
